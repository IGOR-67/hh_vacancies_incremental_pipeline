# mart.salary_baselines

## SQL

```sql
CREATE TABLE mart.salary_baselines AS
SELECT 
    area_id, 
    experience_id,
    ROUND(AVG(
        CASE 
            WHEN salary_from IS NOT NULL AND salary_to IS NOT NULL THEN (salary_from::numeric + salary_to::numeric) / 2
            WHEN salary_from IS NOT NULL THEN salary_from::numeric
            WHEN salary_to IS NOT NULL THEN salary_to::numeric
        END
    )) as base_avg_salary,
    COUNT(*) as total_vacancies
FROM core.fact_vacancies
WHERE salary_currency = 'RUR'
GROUP BY area_id, experience_id;
```

---

## Назначение

Таблица `mart.salary_baselines` — это фундамент аналитической модели.  
Если предыдущая таблица (с навыками) показывала «надстройку»,  
то эта показывает «прожиточный минимум» рынка для конкретных условий.  

Таблица нивелирует влияние географии и стажа.  
Она отвечает на вопрос:  

> «Сколько в среднем платят в этом городе человеку с таким-то опытом,  
> независимо от его специфических навыков?»

Без этой таблицы мы бы сравнивали зарплату Junior-разработчика в Москве  
с зарплатой Senior-мерчендайзера в Барнауле,  
что привело бы к ложным выводам о «стоимости» их навыков.

## Метрики

| Поле | Описание |
|---|---|
| `area_id` + `experience_id` | Контрольные группы; очищают данные от «шума» региональных коэффициентов и карьерных ступеней |
| `base_avg_salary` | Рыночная норма — базовая стоимость специалиста в сегменте |
| `total_vacancies` | Индикатор статистической достоверности сегмента |

### base_avg_salary

| Значение | Вывод |
|---|---|
| Выше базы | Компания платит за редкие навыки или бренд |
| Ниже базы | Масс-маркет либо вакансия с низкими требованиями |

### total_vacancies

| Значение | Вывод |
|---|---|
| `> 50` | Устойчивый рыночный стандарт |
| `< 5` | Зона статистической погрешности; нельзя опираться при долгосрочном планировании |

## Примеры

### area\_id = 88 / between3And6

- **Базовая зарплата:** 152 271 ₽  
- **Вакансий:** 12  
- **Вывод:** Зрелый, стабильный сегмент.  
  База в 152 271 ₽ — «входной билет» для опытного специалиста в данной локации.  
  Любой навык, который изучается в этом сегменте, будет конкурировать с этой цифрой.

### area\_id = 19 / noExperience

- **Базовая зарплата:** 26 813 ₽  
- **Вакансий:** 4  
- **Вывод:** Сегмент старта карьеры.  
  База очень низкая, поэтому даже небольшая доплата  
  (например, +10 000 ₽ за владение Excel) даст огромный `profit_percent`.

---

## Итог

Таблица `salary_baselines` служит **эталоном (бенчмарком)** для оценки эффективности навыков.  
Она позволяет перевести абсолютные значения зарплат в относительные,  
выявляя реальный вклад каждой компетенции в капитализацию специалиста.  
Расчёт произведён на основе агрегированных данных о вилках зарплат  
(среднее значение `salary_from` и `salary_to`),  
что минимизирует субъективность предложений конкретных работодателей.
